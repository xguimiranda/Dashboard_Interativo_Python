## 🎯 Sobre o Projeto

<div align="center">

Este projeto foi desenvolvido durante a **Imersão de Dados com Python** da [Alura](https://www.alura.com.br/), onde exploramos o fascinante mundo da análise de dados salariais na área de tecnologia.

### 🎲 O que o dashboard oferece:

<table>
<tr>
<td width="50%">

**🔍 Filtros Interativos:**
- 📅 **Ano** (2020-2025)
- 👔 **Senioridade** (Junior, Pleno, Senior)
- 📋 **Tipo de Contrato** (Tempo Integral, Freelance, etc.)
- 🏢 **Tamanho da Empresa** (Pequeno, Médio, Grande)

</td>
<td width="50%">

**📊 Métricas e Insights:**
- 💰 **Salário médio** e **máximo**
- 📈 **Total de registros** analisados
- 🎯 **Cargo mais frequente**
- 🏆 **Top 10 cargos** por salário médio
- 📊 **Distribuição salarial** detalhada

</td>

## 🚀 Como Executar Localmente

### 📋 Pré-requisitos

```bash
# Verifique se você tem Python 3.10+ instalado
python --version
```

### 🔧 Instalação Passo a Passo

<div align="center">
<img src="assets/instalacao-local.png" alt="Processo de instalação - Tire uma captura de tela do terminal durante a instalação" width="70%"/>
<br><i>📸 Captura sugerida: Terminal mostrando o processo de instalação</i>
</div>

```bash
# 1️⃣ Clone o repositório
git clone https://github.com/xguimiranda/Dashboard_Interativo_Python.git

# 2️⃣ Navegue até o diretório
cd Dashboard_Interativo_Python

# 3️⃣ Instale as dependências
pip install -r requirements.txt

# 4️⃣ Execute o dashboard
streamlit run app.py
```

### 🌐 Acesso Local

Após executar o comando acima, o dashboard estará disponível em:
- 🔗 **Local:** http://localhost:8501
- 🌐 **Rede:** http://your-ip:8501
